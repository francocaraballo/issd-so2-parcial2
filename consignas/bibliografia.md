# **Bibliografia**

Linux Mint: Proyecto oficial, repositorios y documentación del entorno de escritorio Cinnamon.

Enlace: https://linuxmint.com/

Oracle VM VirtualBox: Manual de usuario, gestión de almacenamiento virtual e integración de utilidades (Guest Additions).

Enlace: https://www.virtualbox.org/manual/

Penguin's Eggs: Repositorio oficial de Piero Proietti, documentación del CLI y empaquetamiento de sistemas (parámetro \--clone).

Enlace: https://github.com/pieroproietti/penguins-eggs

Ventoy: Documentación oficial del gestor de arranque multiboot y especificaciones técnicas del plugin de persistencia (estructuras JSON).

Enlace: https://www.ventoy.net/en/plugin\_persistence.html

Debian live-boot: Manual técnico (manpage) sobre los parámetros del kernel y la sintaxis de persistencia para entornos remasterizados.

Enlace: https://manpages.debian.org/testing/live-boot-doc/live-boot.7.en.html

GParted (GNOME Partition Editor): Documentación oficial sobre gestión de particiones y sistemas de archivos.

Enlace: https://gparted.org/

DDRescue-GUI: Repositorio de Launchpad para la interfaz gráfica de GNU ddrescue.

Enlace: https://launchpad.net/ddrescue-gui

chntpw (Offline NT Password Editor): Documentación del proyecto original para la edición del registro de Windows y manipulación del archivo SAM.

Enlace: http://pogostick.net/\~pnh/ntpasswd/

CGSecurity (TestDisk & PhotoRec): Guías paso a paso para la recuperación de particiones y extracción de datos eliminados.

Enlace: https://www.cgsecurity.org/

Nmap & Zenmap: Referencia oficial del escáner de seguridad y descubrimiento de redes.

Enlace: https://nmap.org/

Wireshark: Guía de usuario para el análisis de protocolos y tráfico de red a bajo nivel.

Enlace: https://www.wireshark.org/

Arduino IDE: Entorno de desarrollo oficial y referencias del lenguaje para pruebas electrónicas.

Enlace: https://www.arduino.cc/en/software

Python 3: Documentación de la biblioteca estándar, específicamente módulos os, subprocess y urllib utilizados para el aprovisionamiento.

Enlace: https://docs.python.org/3/

Especificación Desktop Entry (freedesktop.org): Estándar técnico utilizado para la creación de los accesos directos (.desktop) y categorías de menú.

Enlace: https://specifications.freedesktop.org/desktop-entry-spec/latest/
